# Мессенджер

Простой мессенджер для сдачи в колледже

Будет связь с клиентом через http + websocket для реалтайма

Проект пишется с принципами DDD и чистой архитектурой

Упор на поддержку, читаемость

Авторизация по JWT

Стек
- Spring boot, security, jpa
- nimbus-jose-jwt

### Запуск проекта
1. Скопируйте репозиторий
2. Находясь в папке проекта выполните docker compose up --build
3. Откройте браузер по адресу http://localhost:8080
