# Мессенджер

Простой мессенджер для сдачи в колледже и конечно же для себя

Будет связь с клиентом через http + websocket для реалтайма

Проект пишется с принципами DDD и чистой архитектурой

Упор на поддержку, читаемость

Авторизация по JWT

Стек
- Spring boot, security, jpa, websocket
- nimbus-jose-jwt
